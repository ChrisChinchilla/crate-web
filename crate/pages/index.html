CRATE Think Elastic!

We solve your data scaling problems and make administration a breeze. Easy to scale, simple to use.

What is Crate? Try it!

Powerful

Crate is a powerful, complete backend solution for tabular data, semi-structured records, and binary objects (BLOBs).

Sub-second, super-fast.

Crate is highly available and offers massively parallel access in real-time, enabling sub-second responses on billions of records.

Easy to scale. Simple to use.

Adding nodes on commodity hardware or in the cloud is as simple as a single command in your terminal.

Key Features

  • ease of use
  • SQL interface via HTTP endpoint
  • reliable due to its distributed design
  • elastic / horizontally scalable
  • resilient (self healing) due to automatic sharding and replication
  • highly optimized query engine for distributed group by queries
  • builtin super fast fulltext search
  • builtin blob storage via HTTP endpoint
  • shared nothing architecture means no SPOF
  • web based administration interface

Open Source

The Crate Data open source project was launched in 2013 and today presents the simplest and most powerful option for any developer building a data intensive application. Crate Technology is the company responsible for keeping the momentum behind the Crate project. Crate remains 100% open.

Browse source on Github!

Use Cases

Well Suited

  • storing high volume of "low value" data, such as for IoT applications
  • semistructured/dynamic data
  • operational datastore for web applications that require powerful fulltext search
  • elastic datastore for dynamic startups
  • backend for real time analytics and business intelligence

Not Well Suited

  • systems that require strong transactional consistency, such as banking or accounting
  • strong relational data
