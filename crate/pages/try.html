Try Crate

Get your hands dirty with our Crate Cube!

The Crate Cube is a condensed miniaturized cluster that we’ve custom built to allow you to try out Crate without having to download or install anything! The image above shows three Crate Cubes running in an 12-node cluster configuration with Crate pre-installed in our sophisticated data broom closet*. Just click the green button below to open the Crate Admin Console where you can explore the cluster and run queries against the sample data. The cluster is populated with tweets from the Twitter firehose which are contained in a table named “tweets”. If you click the “Console” button in the left column, you’ll be able to enter SQL queries on the data. For example, to see how many tweets are in the data store, run:

SELECT COUNT(*) FROM tweets;

Give a few queries a try and see how easy it is to query the distributed and auto-sharded data store with SQL.
